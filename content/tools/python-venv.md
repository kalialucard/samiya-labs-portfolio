# Python-venv

> **Type**: Library
> **Language**: Python

**Python-venv** Creation of lightweight 'virtual environments' for Python.

## Official Resources
*   **Official Website**: [https://docs.python.org/3/library/venv.html](https://docs.python.org/3/library/venv.html)
*   **GitHub Repository**: [https://github.com/python/cpython](https://github.com/python/cpython)

## Installation

### Primary Method
```bash
sudo apt install python3-venv
# Usage:
python3 -m venv myenv
```

## Basic Usage
```bash
python-venv --help
```
