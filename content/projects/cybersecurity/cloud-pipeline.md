# Secure Cloud CI/CD Pipeline

Implementing automated security gates within a DevOps workflow.

## Features:
- **SAST/DAST**: Integrated static and dynamic analysis into the build process.
- **Infrastructure as Code**: Hardened AWS environments via audited Terraform modules.
- **Container Security**: Automated scanning of Docker images for known CVEs.
